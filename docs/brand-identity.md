# Nemo.Q 品牌身份规范 (Brand Identity)

## 1. 核心定义
Nemo.Q 是一个手术刀级的智能问数引擎（Surgical-Grade Data Intelligence Engine）。

- **Nemo**: 寓意“潜航者”（Captain Nemo），象征在数据深海中冷静航行的指挥官。
- **.Q**: 代表 **Query** (查询), **Question** (提问), **Quantity** (数据量)。点号（.）体现了技术驱动的精密感。

## 2. 核心价值 (Core Values)
- **精度 (Precision)**: 像手术刀一样精准，拒绝静默错答。
- **透明 (Transparency)**: 所有推理过程、生成的 SQL 和执行路径均可审计。
- **协作 (Collaboration)**: AI 作为一个智慧伙伴，通过澄清与用户深度协同。

## 3. 视觉规范 (Visual Identity)

### 3.1 色彩体系
- **Obsidian Dark**: `#0A0A0A` (背景基调)
- **Surgical White**: `#F9FAFB` (主文字与图标)
- **NovaPulse Blue**: `#3B82F6` (高亮、动效、呼吸光晕)
- **Ghost Gray**: `#262626` (边框、次级文字)

### 3.2 字体
- **主要字体**: `Geist Sans`, `Inter`, system-ui
- **等宽字体**: `Geist Mono`, `JetBrains Mono` (用于 SQL 和数据展示)

### 3.3 Logo 符号
- **概念**: "The Surgical Lens"
- **形态**: 一个正圆环，右下角有一个 45 度、略微偏移的切削短划。
- **动效**: 在系统思考时，Q 的中心会产生频率为 0.5Hz 的 NovaPulse 蓝色光晕。

## 4. 品牌语调 (Tone of Voice)
- **冷静 (Calm)**: 不使用夸张的修辞，不承诺做不到的事情。
- **专业 (Professional)**: 使用准确的技术术语。
- **高效 (Efficient)**: 回复直接触达核心，减少冗余社交辞令。

## 5. Slogan
- **EN**: *Precision In, Truth Out.*
- **CN**: *精准问询，还原真相。*
